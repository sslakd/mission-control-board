### GIAI ĐOẠN 1: Test Sản Phẩm (15/06 - 15/08) — 2 tháng

**Mục tiêu:** Chốt 1 sản phẩm đầu tiên, đặt hàng mẫu, test thị trường

| Tháng | Việc cần làm | Thời gian |
|-------|-------------|-----------|
| **Tháng 6** | • Nghiên cứu 3 sản phẩm tiềm năng bằng Helium 10 | ~20h/tháng |
| (15/06-15/07) | • Kiểm tra trademark (Chương 4 - trang 38) | |
| | • Tính toán chi phí ROI cho từng sản phẩm | |
| | • Liên hệ supplier (Alibaba, 1688) — hỏi giá mẫu | |
| **Tháng 7** | • **Đặt mẫu** 1-2 sản phẩm (order mẫu về Việt Nam) | ~20h/tháng |
| (15/07-15/08) | • Kiểm tra chất lượng, chụp ảnh sản phẩm | |
| | • Đăng ký brand (USPTO or Amazon Brand Registry) | |
| | • Xin miễn GTIN nếu cần (Chương 5) | |
| | • Viết listing: title, bullet points, description (Chương 5) | |

**Chi phí ước tính giai đoạn 1:**
- Mẫu sản phẩm: ~$100-200 (ship từ TQ về VN)
- Helium 10: ~$79/tháng (nếu mua, có thể dùng free 1 tháng)
- Amazon fee: $39.99/tháng × 2 = $80
- **Tổng giai đoạn 1: ~$300-400**

> **⚠️ Lưu ý:** Đây là giai đoạn QUAN TRỌNG NHẤT. Đừng vội. Chọn sai sản phẩm = chết. Học từ cẩm nang: "100 ý tưởng → 15-20 tiềm năng → 3-5 có thể triển khai"

### GIAI ĐOẠN 2: Launch Sản Phẩm Đầu Tiên (15/08 - 15/10) — 2 tháng

**Mục tiêu:** Đặt hàng số lượng >100, gửi FBA, launch, có sale đầu tiên

| Thời gian | Việc cần làm |
|-----------|-------------|
| **15/08 - 15/09** | • Đặt hàng số lượng lớn (100-300 units) từ supplier |
| | • Chuẩn bị bao bì, nhãn mác (FBA-compatible packaging) |
| | • Chụp ảnh sản phẩm chuyên nghiệp (tối thiểu 6 ảnh) hoặc ảnh AI-enhanced |
| | • Tìm forwarder / đơn vị logistics vận chuyển VN → Mỹ |
| | • **Gửi hàng sang kho Amazon FBA** (Chương 6) |
| **15/09 - 15/10** | • Hàng đến kho Amazon + check-in |
| | • Thiết lập chiến dịch quảng cáo Sponsored Products (Chương 7) |
| | • **Launch sản phẩm** — chạy PPC với bid thấp ban đầu |
| | • Apply Amazon Vine (nếu đã đăng ký Brand Registry) — lấy 15-30 reviews đầu |
| | • Theo dõi tồn kho, điều chỉnh quảng cáo (Chương 8) |

**Chi phí ước tính giai đoạn 2:**
- Hàng hóa 100-300 units: ~$800-2,000 (tùy sản phẩm)
- Vận chuyển VN → Mỹ: ~$300-600
- Chụp ảnh sản phẩm: ~$100-200
- Amazon fee + Vine fee: ~$200
- PPC ngân sách 2 tuần đầu: ~$200
- **Tổng giai đoạn 2: ~$1,500-3,000**
